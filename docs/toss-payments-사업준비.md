> 토스페이먼츠는 초기 가입비/보증금/보증보험 없이 거래 수수료만 발생.

## PG 수수료 (거래 시 자동 차감)

| 결제 수단 | 수수료율 |
|-----------|----------|
| 신용/체크카드 | 약 2.5~3.5% |
| 계좌이체 | 약 1.5~2.0% |
| 가상계좌 | 건당 200~300원 |
| 간편결제 (토스페이 등) | 약 2.5~3.5% |

- 수수료에 부가세 10% 별도 (예: 수수료 3% → 실제 3.3%)
- 정산 시 자동 차감, 별도 납부 없음
- 정산 주기: D+2~D+7 영업일
- 연매출 규모에 따라 수수료율 협상 가능

## 준비 단계 (순서대로)

### 1단계: 사업자등록

- **방법**: 홈택스 온라인 신청 또는 관할 세무서 방문
- **필요 서류**: 신분증, 임대차계약서 (자택이면 자가 증빙)
- **사업자 유형**: 간이과세자
- **소요 시간**: 당일~3일
- **비용**: 무료

### 2단계: 사업용 통장 개설

- **방법**: 사업자등록증 지참 후 은행 방문
- **용도**: 토스페이먼츠 정산금 수령 계좌
- **비용**: 무료

### 3단계: 토스페이먼츠 가입 & 테스트키 발급

- **사이트**: tosspayments.com
- **참고**: 테스트키는 사업자 심사 전에도 발급 가능 → 개발 먼저 진행 가능

### 4단계: 통신판매업 신고

- **방법**: 정부24 온라인 신청
- **필요 서류**: 사업자등록증, 구매안전서비스 이용확인증 (토스페이먼츠 가입 후 발급)
- **소요 시간**: 3~7일
- **비용**: ~40,200원

### 5단계: 가맹점 심사 제출

- **제출 서류**:
  - 사업자등록증
  - 대표자 신분증
  - 사업용 통장 사본
  - 서비스 URL (테스트 사이트도 가능)
- **소요 시간**: 심사 1~3영업일
- **비용**: 무료

## 진행 플로우

```
신분증 준비
  → ① 사업자등록 (홈택스/세무서, 무료)
    → ② 사업용 통장 개설 (은행, 무료)
      → ③ 토스페이먼츠 가입 & 테스트키 발급
        → ④ 통신판매업 신고 (정부24, ~4만원)
          → ⑤ 가맹점 심사 제출 → 승인 → 라이브키 전환
```

> 개발은 ③ 테스트키 발급 시점부터 병렬로 진행 가능.
