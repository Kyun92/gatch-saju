# 캐릭터 픽셀 아바타 생성 프롬프트

## 생성 순서

> **반드시 Step 1 → Step 2 순서로 진행**
> 1. 레퍼런스 이미지 + 프롬프트로 기본 베이스 남/여 2종 생성
> 2. 베이스 이미지를 첨부하고 각 오행 변형 프롬프트 실행

---

## Step 1: 베이스 스프라이트 생성

### 공통 규칙

```
스타일: 모던 픽셀 일러스트 (일본 픽셀아트 감성)
해상도: 256×256px 이상으로 생성 → 후처리에서 64×64로 리사이즈
배경: 단색 베이지 (#F5E6C8)
시점: 정면
비율: 3~3.5등신 SD 캐릭터
색감: 자유 (부드럽고 따뜻한 톤)
셰이딩: 디더링 + 부드러운 명암, 픽셀 경계 선명
특징: 머리카락 결 표현, 옷 질감, 바닥에 작은 파티클 이펙트
```

## 이미지 후처리

1. 생성된 이미지를 정확히 **64×64px**로 리사이즈 (Nearest Neighbor 필수!)
2. 배경(베이지)을 **투명**으로 처리
3. **PNG** 형식으로 저장
4. 선택: 128×128 @2x 버전도 함께 저장 (Nearest Neighbor 업스케일)

## 품질 체크리스트

- [ ] 모든 10종이 동일한 캔버스 크기 (64×64)
- [ ] 레퍼런스 이미지와 동일한 그림체 유지
- [ ] 베이스와 동일한 등신 비율 (3~3.5등신)
- [ ] 투명 배경으로 어두운/밝은 배경 모두에서 자연스러움
- [ ] 오행별 색상이 명확히 구분됨 (초록/빨강/갈색/은색/파랑)
- [ ] 남녀 캐릭터가 시각적으로 구분됨
- [ ] 스타일이 10종 전체에서 일관됨 (베이스에서 파생된 느낌)
- [ ] 64×64에서 캐릭터가 캔버스의 70% 이상 차지
- [ ] 디더링 셰이딩, 선명한 픽셀 경계 유지
