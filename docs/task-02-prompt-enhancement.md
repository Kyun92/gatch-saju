## 프롬프트 개선 상세

### 기존 (7섹션, 3000-4000자)
인사 → 기질/재능 → 재물/직업 → 인간관계 → 건강 → 현재시기 → 실천가이드

### 개선 (12섹션, 6000-8000자)
1. 인사 및 전체 요약 (생동감 있는 비유)
2. 사주 원국 분석 (사주팔자 상세)
3. 오행 분석 (분포 테이블)
4. 일주 분석 (성격 심층)
5. 팩트 폭격 (솔직한 조언)
6. 따뜻한 위로
7. 성격 분석 (외유내강 등)
8. 직업운 (적성 테이블)
9. 재물운 (재물 전략 테이블)
10. 이성운 / 인간관계
11. 건강 / 방위 / 거주
12. 종합 개운법 (행운 가이드 테이블)

## 신규 HTML 컴포넌트

| 클래스 | 용도 | 스타일 |
|--------|------|--------|
| `.frank-box` | 팩트폭격/솔직한 조언 | 빨간색 테두리 박스 |
| `.warmth-box` | 위로/개운 섹션 | 골드색 따뜻한 박스 |
| `.fortune-table` | 데이터 테이블 (오행, 적성 등) | 다크 테마 테이블 |

## 핵심 개선점
- 3체계 교차분석 (사주 + 자미두수 + 서양점성술) 필수
- 비유와 스토리텔링 강화 ("화려한 조명 속 고독한 아티스트" 스타일)
- 명리학적 해석 + 현실 조언 이중 구조
- 따뜻하지만 솔직한 톤 (김창욱 스타일)
- Opus 모델 사용으로 품질 향상
