## 라우트 동작

| 경로 | 비로그인 | 로그인 (프로필 미완) | 로그인 (프로필 완료) |
|------|----------|---------------------|---------------------|
| `/login` | 200 (로그인 페이지) | 200 | 200 |
| `/onboarding` | 302 → /login | 200 (생년월일 입력) | 200 |

## 사용자 여정

```
랜딩(/) → "시작하기" 클릭 → /login
→ 카카오/네이버/구글 선택 → OAuth 인증
→ /onboarding (생년월일 입력) → 차트 생성
→ /payment (결제) → 결과 조회
→ /dashboard (마이페이지: 내 풀이, 구독 관리)
```
