# Task 06: MBTI 통합 — 사주 × 자미두수 × 서양점성술 × MBTI 교차 해석

## 배경

현재 서비스는 사주팔자, 자미두수, 서양점성술 3개 체계를 교차 분석하여 운세를 생성한다. 여기에 MBTI를 4번째 축으로 추가하되, **별도 섹션이 아닌 기존 풀이 안에 자연스럽게 녹여넣는 방식(A안)**으로 통합한다.

### 왜 MBTI인가

- 한국 2030 세대에서 MBTI는 자기소개 필수 항목 수준의 인지도
- 입력 부담이 낮음 (4글자 선택 추가)
- SNS 공유율이 높은 콘텐츠 유형
- 세 가지 명리 체계와 교차하면 "왜 내가 이 MBTI인지"를 운명적으로 설명해주는 독특한 콘텐츠가 됨

### A안을 선택한 이유 (vs B안 별도 리포트)

| 비교 항목 | A안: 보조 렌즈 | B안: 별도 콘텐츠 |
|:---------:|:-------------:|:---------------:|
| 분량 증가 | 최소 (섹션당 1~2문장) | 대폭 (새 섹션 추가) |
| 모순 관리 | 자연스러운 프레이밍 가능 | 별도 해명 필요 |
| 개발 공수 | 낮음 (프롬프트 수정 위주) | 중간 (새 템플릿 필요) |
| 서비스 정체성 | 명리 중심 유지 | 정체성 희석 우려 |
| 토큰 비용 | 거의 동일 | 추가 호출 필요 |

## 변경 범위

### 1. DB 스키마 변경

**파일:** `src/db/schema.ts`

`users` 테이블에 MBTI 컬럼 추가:

```
mbti TEXT (nullable)
```

- 가입 시 선택 입력 (필수 아님)
- 유효값: 16가지 MBTI 유형 (INFP, ENTJ 등)
- null인 경우 MBTI 교차 해석 없이 기존 3체계로만 풀이

### 2. API 변경

**파일:** `src/api/routes/chart.ts`, `src/api/routes/reading.ts`

- 사용자 등록/수정 API에 `mbti` 필드 추가
- 운세 생성 시 사용자의 MBTI를 프롬프트에 포함

### 3. 프롬프트 변경 (핵심)

**파일:** `src/fortune/prompts.ts`

#### 3-1. COMPREHENSIVE_SYSTEM (종합 사주풀이) 수정

현재 12개 섹션 구조를 유지하되, 시스템 프롬프트에 다음 지침을 추가:

```
[MBTI 교차 해석 지침]
사용자의 MBTI가 제공된 경우, 각 섹션에서 사주/자미두수/점성술 해석과
MBTI 특성이 겹치는 지점을 1~2문장으로 자연스럽게 연결하라.

적용 원칙:
1. MBTI를 별도 섹션으로 분리하지 말 것
2. 각 섹션 안에서 "양념"처럼 녹여넣을 것
3. 세 체계의 해석과 MBTI가 일치하는 경우:
   → "태어나면서부터 이 기질을 타고난 구조"로 연결
4. 세 체계의 해석과 MBTI가 모순되는 경우:
   → "타고난 기질 vs 후천적 성격의 간극"으로 프레이밍
5. 한자/전문 용어 대신 비유 체계를 사용할 것
```

**섹션별 MBTI 적용 가이드:**

| 섹션 | MBTI 활용 방향 | 예시 |
|:----:|:--------------|:-----|
| 1. 전체 요약 | 사주 기질과 MBTI의 일치/불일치를 첫 인상으로 제시 | "깊은 바다 같은 기운을 타고난 당신이 INFP인 건 우연이 아닙니다" |
| 4. 일주 분석 | 일주 성향과 MBTI 주기능(Fi/Te 등) 연결 | "壬寅일주의 관조적 성향이 INFP의 내향 감정(Fi)과 시너지를 이룹니다" |
| 5. 팩트 폭격 | MBTI 약점과 사주 약점이 겹치는 패턴 지적 | "INFP의 우유부단함 + 화 기운 부재 = 실행력 부족이 태생적 구조" |
| 7. 성격 분석 | 타고난 성격(사주) vs 표현 성격(MBTI) 비교 | "사주로는 외유내강인데 ENFJ로 표현되는 건 후천적 적응의 결과" |
| 8. 직업운 | MBTI 적성과 사주 적성의 교집합 강조 | "문곡성 + INFP = 글쓰기/상담 분야에서 교집합" |
| 10. 인간관계 | MBTI 궁합론과 사주 귀인론 교차 | "ENFJ가 귀인인 이유는 사주에서 부족한 화 기운을 보충해주기 때문" |

#### 3-2. DAILY_SYSTEM (일일운세) 수정

일일운세 프롬프트에 MBTI 관련 지침 추가:

```
사용자의 MBTI가 제공된 경우, 오늘의 실천 팁을 MBTI 성향에 맞게 조정하라.
예: INFP → "혼자만의 시간이 보약인 날", ENTJ → "리더십을 발휘할 기회가 오는 날"
분량은 1~2문장 추가에 그칠 것.
```

### 4. 일일운세 생성 로직 변경

```typescript

사용자 메시지에 MBTI 정보 포함:

```

## 한자 제거 — 비유 체계 도입

MBTI 통합과 동시에, 젊은 층 타겟에 맞춰 한자 용어를 비유 체계로 치환한다.

### 프롬프트에 추가할 비유 매핑 지침

```
[비유 체계 지침]
사주/자미두수/점성술의 전문 용어를 직접 사용하지 말고,
다음과 같은 비유 체계로 치환하여 작성하라.

오행 비유:
- 금(金) → "쇠 기운" 또는 "날카로운 에너지"
- 목(木) → "나무 기운" 또는 "성장 에너지"
- 수(水) → "바다/물 기운" 또는 "깊은 에너지"
- 화(火) → "불꽃 기운" 또는 "열정 에너지"
- 토(土) → "흙/대지 기운" 또는 "안정 에너지"

연도 비유:
- 간지 연도 → 동물+색 조합 (예: 丙午년 → "붉은 말의 해")

자미두수 비유:
- 주성 이름 대신 역할 설명 (예: 文曲 → "문학과 예술의 별")
- 궁위 이름 대신 생활 영역 (예: 재백궁 → "재물 하우스")

점성술:
- 하우스 번호 대신 생활 영역 (예: 10하우스 → "커리어 하우스")
- 애스펙트 → 관계 표현 (예: 스퀘어 → "긴장 관계")

한자는 괄호 안에 보조적으로만 사용 가능.
예: "깊은 바다 타입(壬水)"은 OK, "壬水 일간"은 NG.
```

## 풀이 품질 기준

### MBTI 교차 해석이 잘된 예시

> 당신의 타고난 기운은 '깊은 바다'예요. 끝이 보이지 않는 수면 아래에 상상할 수 없을 만큼 복잡하고 풍부한 내면 세계가 펼쳐져 있죠. 여기에 커리어 하우스의 '문학과 예술의 별'이 빛나고 있고, 게자리 상승궁이 감정의 갑옷을 씌워주고 있어요. **INFP라는 기질은 이 세 가지 체계가 이미 가리키고 있던 방향의 자연스러운 귀결입니다.** 당신은 태어나면서부터 INFP일 수밖에 없는 운명적 구조를 타고난 셈이죠.

### MBTI 교차 해석이 잘못된 예시 (피해야 할 것)

> 당신의 사주를 보면 壬水 일간이고 INFP입니다. INFP는 이상주의적이고 창의적인 유형입니다. 사주에서도 비슷한 점이 보입니다.

**문제점:** 단순 나열, 교차 분석이 아닌 병렬 배치, 한자 직접 사용

## 체계 간 모순 처리 전략

사주/자미두수/점성술은 같은 생년월일시에서 출발하므로 해석이 대체로 수렴하지만, MBTI는 자기 보고식이라 모순이 발생할 수 있다.

| 상황 | 프레이밍 전략 |
|:----:|:------------|
| 사주 "내향적" + MBTI "E" | "타고난 기질은 내향적이지만, 후천적으로 외향적 에너지를 발달시킨 경우. 혼자 충전한 뒤 밖으로 나가는 패턴" |
| 사주 "감성적" + MBTI "T" | "감성이 풍부하지만, 그것을 논리적 프레임으로 정리하는 방식을 선택한 사람. 분석의 출발점이 감정" |
| 사주 "리더형" + MBTI "P" | "리더십의 자질은 있지만, 구조보다 유연한 방식을 선호. 카리스마형이 아닌 영감형 리더" |
