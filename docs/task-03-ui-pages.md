## 랜딩 페이지 구성

1. **네비게이션** - 로고 + 메뉴 (서비스 소개, 이용방법, 가격, 시작하기)
2. **히어로 섹션** - 3개 궤도 오브 애니메이션 (사주/자미두수/서양점성술 시각화)
3. **통계 바** - 서비스 수치
4. **왜 3중 분석인가** - 벤 다이어그램 시각화
5. **3대 시스템 카드** - 사주팔자, 자미두수, 서양점성술 각각 설명
6. **이용 방법** - 3단계 (정보입력 → AI 분석 → 결과 확인)
7. **후기**
8. **가격** - 종합 사주풀이 (₩29,900) / 매일 운세 구독 (₩9,900/월)
9. **최종 CTA**
10. **푸터**

## 등록 페이지

- 이름, 전화번호 (자동 포맷), 생년월일, 출생시간, 출생도시 (20개 도시), 성별
- 클라이언트 유효성 검증 + 한국어 에러 메시지
- POST /api/users 호출 → 성공시 /payment 리다이렉트
- ARIA 접근성 지원

## 결제 페이지

- 플랜 선택 (일회/구독) 라디오 카드
- 결제 수단 탭 (신용카드, 카카오페이, 네이버페이, 토스)
- Toss Payments SDK 연동 포인트 준비
- 주문 요약 사이드바

## 디자인 특성

- 다크 우주 테마 (--bg-deep: #0d0d1a, 금색 강조)
- 3개 시스템별 고유 컬러: 사주(파랑), 자미두수(보라), 서양(로즈)
- IntersectionObserver 스크롤 애니메이션
- 모바일 반응형 (900px, 600px 브레이크포인트)
- 외부 JS 의존성 없음 (Pretendard 폰트만 CDN CSS)
